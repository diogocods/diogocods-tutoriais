Instalando o GCC para linguagem C e C++ no linux - Passo a Passo

O que iremos fazere é instalar o pacote build-essential, e então testar o compilador gcc criando uma aplicação básica em linguagem C. O pacote build-essential é, na verdade, uma referência (lista) para todos os pacotes que são necessários para compilar programas (pacotes) no Debian. Ele inclui diversos elementos, tais como o compilador gcc (que testaremos aqui), compilador g++ (para C++), a ferramenta make, diversas bibliotecas e utilitários.

Para instalar o pacote build-essential no Ubuntu (e derivados) basta usar o gerenciador de pacotes apt – lembrando que é necessário estar logado como root ou possuir privilégios de usuario administrador (sudo):

$ sudo apt update
$ sudo apt install build-essential

Vamos verificar as versões instaladas se o pacote foi instalado corretamente:

$ gcc --version

No seu terminal, a resposta será semelhante a esta.

Agora iremos escrever e compilar um pequeno programa em linguagem C para testar o funcionamento do gcc: (Passo a Passo e com imagens )

Vamos criar o arquivo (use seu editor de textos favorito; usaremos o nano aqui):

$ nano teste.c

Digitar o seguinte conteúdo no arquivo:

#include < stdio.h >
int main() {
printf("diogocods. Treinamentos em Tecnologia!\n");
return 0;
}

Após criar o arquivo fonte e codificá-lo, vamos salvá-lo e então o fecharemos para voltar ao terminal. O próximo passo é compilar o arquivo com o comando a seguir:

$ gcc teste.c -o teste

Será criado o arquivo binário teste no diretório onde o comando foi executado. Vamos rodar o programa para testar:

$./teste

Saída: diogocods treinamentos em tecnologia.
